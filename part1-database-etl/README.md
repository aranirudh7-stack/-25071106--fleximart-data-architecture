# Part 1: Database ETL Pipeline

This module implements the ETL (Extract, Transform, Load) process for the FlexiMart project.

## Steps Covered
- Extract raw data from CSV files
- Transform data by cleaning and standardizing values
- Load processed data into MySQL relational tables

## Files
- etl_pipeline.py : Python ETL script
- schema_documentation.md : Database schema details
- business_queries.sql : SQL queries for business analysis
- data_quality_report.txt : Data quality issues and resolutions
