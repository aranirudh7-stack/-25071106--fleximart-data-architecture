# Part 2: NoSQL (MongoDB)

This module demonstrates the use of MongoDB for handling semi-structured product catalog data in the FlexiMart project.

## Objectives
- Store product catalog data in MongoDB
- Perform CRUD operations
- Use aggregation pipelines for analysis

## Files
- products_catalog.json : Sample product catalog data
- mongodb_operations.py : Python script for MongoDB operations
- nosql_analysis.md : Explanation of NoSQL design choices
